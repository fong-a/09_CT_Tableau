Tableau Public Bar Chart Tutorial

Creating a Bar Chart in Tableau Public

  1. Open Tableau Public and import your billionaire dataset CSV file.
  2. Move to "Sheet 1" to start creating your visualization.
  3. Drag 'personName' to Rows and 'finalWorth' to Columns to create a bar chart.
  4. Sort the bars in descending order and format the 'finalWorth' values for clarity.
  5. Use the "Color" mark to categorize bars and add labels for better readability.
  6. Add descriptive titles and save your workbook to Tableau Public.
